Adapter - Дескриптор видеокарты.
Device - Абстракция над видеокартой у которой есть заданные features (adapter.features()).
Buffer - большой объём данных пересылаемых на GPU. Буффер непрерывный, обычно сипользуется для хранения простых
вещей (структуры, массивы), пример буфера - вершинный буфер
Buffer Layout - описывает структуру буфера, то есть что там лежит. Сам layout передаётся в пайплан, что бы конвейер знал
как обрабатывать данные из приходящего буфера.

RenderPipeline - шейдерный конвейер (There are 3 main types of shader: vertex, fragment, and compute)

Fragment shader - каждый пиксель в результирующем изображении получает по крайней мере один фрагмент

Как работает surface в связке с видеокартой:
Surface предоставляет на каждом проходе рендера дескриптор текстуры, который использует сам рендер пас для записи в него
информации которая была получена во время выполнения команд при проходе рендер пасса (результат работы конвейера шейдеров
в том числе).
